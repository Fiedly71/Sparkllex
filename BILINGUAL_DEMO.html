3. How to Share Language-Specific Links

Scenario: You want staff to see the site in Spanish

What you do:

1. Visit the page you want to share 2. Add ?lang=es to the URL 3. Send that link to your staff member Example: yoursite.com/signup.html?lang=es

What happens:

  1. Staff member clicks your Spanish link
  2. Page loads in Spanish
  3. System saves lang=es to their localStorage
  4. They navigate to any other page on your site
  5. The page stays in Spanish (no ?lang= needed!)

4. How Translation Works in HTML

Step 1: Add an ID to your HTML element

<h1 id="pageTitle">Welcome</h1> <button id="submitBtn">Submit</button> <input id="emailField" placeholder="Email">

Step 2: Define the translation in translations.js

const translations = { en: { pageTitle: "Welcome", submitBtn: "Submit", emailField: "Email" }, es: { pageTitle: "Bienvenido", submitBtn: "Enviar", emailField: "Correo" } };

Step 3: That's it! The system handles the rest

When the page loads:

  • ‚úì apply-translations.js detects the language
  • ‚úì Finds all elements with IDs
  • ‚úì Looks up each ID in the translations
  • ‚úì Updates the element's text automatically
