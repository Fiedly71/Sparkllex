# Sparkllex Website Project

## Project Overview
Sparkllex is a comprehensive platform providing membership, booking, and administrative services.

## Ownership & License
**Sparkllex** is the sole owner of the source code contained in this repository. All rights reserved.

## Project Structure

### 01_MARKETING
Marketing and public-facing pages:
- `index.html` - Homepage
- `how-it-works.html` - Feature explanation
- `pricing.html` - Pricing information
- `signup.html` - User registration
- `success.html` - Registration success page

### 02_MEMBERS_APP
Member portal and services:
- `membership-status.html` - Member account status
- `book-service.html` - Service booking interface
- `history.html` - Member booking history
- `support-ia.html` - AI-powered support

### 03_OPERATIONS
Operations management tools:
- `agenda.html` - Team schedule management
- `team-manager.html` - Team administration
- `crm-clients.html` - Client relationship management

### 04_ADMIN_METRICS
Administrative dashboards:
- `executive-finance.html` - Financial metrics and analytics
- `expansion.html` - Expansion metrics and KPIs

## Root Files

### Configuration Files
- `styles.css` - Global styles with Tailwind CSS configuration
- `config.js` - Application configuration and settings
- `README.md` - Project documentation (this file)

## Getting Started

1. Ensure all HTML5 files are properly linked to `styles.css`
2. Configure API endpoints in `config.js`
3. Customize styling in `styles.css` as needed

## Development Notes

- All HTML files use semantic HTML5 structure
- Responsive design principles are applied throughout
- Professional naming conventions are maintained
- All files follow accessibility best practices
